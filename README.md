# 🔢 Number Analyzer System (Python)

## 📌 Description
This project is a menu-driven Python application that analyzes numbers and performs various mathematical operations. It allows users to check different properties of a number such as prime, Armstrong, perfect, and palindrome, along with additional operations like reversing a number and calculating the sum of its digits.

The project is built using modular programming, where the main program handles user interaction and a separate file contains all the logic functions.

---

## 🎯 Objectives
- To implement number-based algorithms
- To understand menu-driven programming
- To practice modular programming using multiple files
- To handle user input using exception handling

---

## 📂 Project Structure
```
main.py                         # Main menu, user input, and function calls
NumberAnalayzerSystem2.py       # Contains all number-related functions
README.md
```

---

## 🔄 Program Flow
1. The user selects an operation from the menu  
2. The program takes input number  
3. The corresponding function is called from the utility file  
4. The result is displayed
